Why This Website Exists

I'm tired of super flashy websites where the content is buried under animations, pop-ups, cookie banners, and tracking scripts.

I believe we need to go back to the basics and re-learn how to build websites that are:

  • Clear
  • Accessible
  • Straight to the point

This website is a tiny protest against the bloated, over-engineered web.

Welcome Back to 1995

When you land here, I want you to feel like you've stepped into the early internet:

  • No dark patterns
  • No infinite scroll
  • No cookie banners
  • Just text, links, and ideas

What This Project Stands For

This project stands for simplicity, clarity, and accessibility. It is a reminder that the web can be a place for straightforward communication without unnecessary distractions.

What This Site Is

This site is a collection of HTML pages that demonstrate how to create a functional and user-friendly website using only HTML. There are no stylesheets or scripts, just pure HTML.

What I Hope You Get From This

I hope this site inspires you to think differently about web design and development. Sometimes, less is more. By focusing on content and usability, we can create better experiences for users.

Where to Next?

If you're interested in exploring more about simple web design, consider checking out resources on HTML best practices, accessibility guidelines, and minimalist design principles.
